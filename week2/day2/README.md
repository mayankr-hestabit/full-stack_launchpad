# FAQ Accordion

A simple and responsive FAQ Accordion built using HTML, CSS, and JavaScript. Users can expand or collapse questions by clicking on them, providing a clean and interactive FAQ section.

---

## Features

- Responsive Design
- Expand/Collapse FAQ
- Smooth Toggle Animation
- Rotating Arrow Icon
- Clean UI
- Beginner Friendly Code

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

---

## How It Works

- JavaScript selects all FAQ questions.
- A click event is added to every question.
- Clicking a question toggles the answer.
- The arrow icon rotates when opened.
- Previously opened FAQ closes automatically.

---

## Concepts Used

- querySelectorAll()
- addEventListener()
- nextElementSibling
- classList.toggle()
- forEach()
- DOM Manipulation
