# Day 3 — Git Mastery

## Calculator Module
A simple calculator built across 8 commits to demonstrate git bisect,
revert, stash, and merge conflict resolution.

## Functions
- `add(a, b)` — returns sum of a and b
- `subtract(a, b)` — returns a minus b
- `multiply(a, b)` — returns product of a and b
- `divide(a, b)` — divides a by b
- `power(a, b)` — raises a to the power of b
- `validate(a, b)` — throws error if inputs are not numbers
- `absolute(a)` — returns absolute value of a

## Usage
```bash
node test.js
```

## Git Operations Practiced
- `git bisect` — binary search to find buggy commit
- `git revert` — undo a commit without rewriting history
- `git stash` — temporarily save uncommitted work
- Merge conflict resolution
